# Yaak 桌面 API 客户端

Yaak 是一个快速、注重隐私的桌面 API 客户端，支持 REST、GraphQL、SSE、WebSocket 和 gRPC。
项目基于 Tauri、Rust 和 React 构建，强调轻量、高性能与本地优先。

本仓库目前由本项目独立维护，后续将持续进行二次开发。

## 核心特性

- 离线优先：本地可用，减少对云端依赖。
- 协议丰富：支持 REST、GraphQL、gRPC、WebSocket、SSE。
- 安全可靠：支持 OAuth 2.0、JWT、Basic Auth 等认证方式，敏感信息可加密存储。
- 组织协作：支持工作区、文件夹、环境变量，以及与 Git 工作流协同。
- 可扩展：支持插件机制、模板标签与主题定制。
